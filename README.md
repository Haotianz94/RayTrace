#RayTrace
A RayTracer for 2D Image Rendering

##Features
 * Used Phone's model
 * Used C++
 * Developed on Visual Studio 2012

##Accomplished Functions
###Basic Functions
 * Diffusion
 * Highlight
 * Reflection
 * Refraction
###Enhanced Functions
 * Soft Shadow
 * Texture
 * Anti-alisa
 * KD-tree with Triangle Mesh Rendering
 * Random Diffusion
 * Depth of Field
 * Triangle Mesh Rendering with Texture
